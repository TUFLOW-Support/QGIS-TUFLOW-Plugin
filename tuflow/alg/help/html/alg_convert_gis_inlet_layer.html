Convert GIS Inlet Layer to SWMM

This tool converts a GIS layer with 2D connectivity information based upon a mathematical discharge curve and inlet elevation data into SWMM curves (inp file) and an inlet usage layer that can be used in a TUFLOW-SWMM model. This kind of data can be exported from an XPSWMM model or other models using the same mathematical relationship for inlet discharges.

The tool works on discharges computed from the equation: Q=coefficient*depthexponent
