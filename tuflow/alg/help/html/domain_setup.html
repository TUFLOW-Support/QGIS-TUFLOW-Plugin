Domain Setup

The canvas reflects the map window canvas and will be updated with added/removed layers, changed visibility, and styling.

  • - Toggle on to start drawing or to edit an already drawn domain.
  • - Clear the drawn domain.
  • - Zoom canvas to full extent.
  • - Create a domain based on the extent of a given layer. Input angle will be retained.
  • - Rounds the domain values (does not round the angle)
  • - Create and copy domain TUFLOW commands to the clipboard.
